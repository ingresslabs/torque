# Atlassian Torque Case Study Attachments

These files support `blog-atlassian-torque-case-study.html`.

The stack and values files are sanitized. Secret material is represented by
`secret://` references or Kubernetes `secretKeyRef` fields. Do not replace these
files with resolved Secrets, kubeconfigs, cluster logs, or local provider data.

The report files are generated verifier and Helmer HTML and JSON outputs from
the lab run. They are intended as review evidence for the case study, not as a
reusable production baseline. Do not publish unsanitized reports for this
case-study bundle.
